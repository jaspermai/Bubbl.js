API DOCUMENTATION

VideoCanvas

A video canvas represents a fixed canvas on the screen that will hold related videos. Each VideoCanvas object will have its own functions to create video bubbles, add links, and remove video bubbles. There can be multiple video canvases on one webpage, as demonstrated in the home page examples.

Type Parameters

Name Type Default Description
divIdName String body An existing ID of the div that you want to create the VideoCanvas in. (ie. to create a VideoCanvas inside of a div with id="block", you should pass in "block")
width String '100%' Width of the VideoCanvas that encapsulates the videos.
height String '100%' Height of the VideoCanvas that encapsulates the videos.
color String 'transparent' Background color of the VideoCanvas. Takes in standard color codes.
isFilter Boolean true Determines if you want the videos in the VideoCanvas to have a tinted filter overlay. The tinted filter gives a more animated bubble look.

Constructor

Create a VideoCanvas by calling the VideoCanvas constructor. 

                    
            new VideoCanvas(divIdName, width, height, color, isFilter)

Constructor Examples 

                    
            const vidCanvas = new VideoCanvas('video-block', ‘80%’, ‘900px’, ‘#75ddb0’, true)
            const vidCanvas2 = new VideoCanvas('second-block')

Functions

After creating a Video Canvas object, you can access its several functions to insert and customize your videos.

addVideo

  • description: Adds a new video bubble with videoId to your VideoCanvas. This video will be fully interactive in standard bubble mode and full-screen mode.
  • return value: void
  • arguments:
    Name Type Default Description
    videoId String REQUIRED A new, unique id to access your video bubble.
    src String REQUIRED The file path to your video.
    videoSize String: {'small', 'medium', 'large'} 'medium' A control variable that decides the size of the video bubble. Possible options are ‘small’, ‘medium’ and ‘large’.
    videoVolume Number: [0,1] 1 A control variable that decides the adjusted volume of the video. Possible options are decimal values between 0 and 1, inclusive.
  • examples: 
                                
                vidCanvas.addVideo('vid1', 'examples/vids-creator/youtube1.mp4', 'medium', 0.8)
                vidCanvas.addVideo('vid3', 'examples/vids-creator/youtube3.mp4')

videoLink

  • description: Attaches a video link that is directly accessible through the video bubble. This link should be associated with the video.
  • return value: void
  • arguments:
    Name Type Default Description
    videoId String REQUIRED An existing id of the video bubble that you want to attach an external link to.
    url String REQUIRED The external link that will be attached to the video bubble.
  • examples: 
                                
                vidCanvas.videoLink('vid1', 'https://www.youtube.com/watch?v=qPT-Y00_HlE')

removeVideo

  • description: Removes an existing video bubble with videoId from the VideoCanvas.
  • return value: void
  • arguments:
    Name Type Default Description
    videoId String REQUIRED An existing id of the video bubble that you want to remove.
  • examples: 
                                
                vidCanvas.removeVideo('vid1')

Interactive Features

Here are a few of the interactive features that Bubbl.js provides.

  • Standard Bubble Mode
    • Play on hover, pause off hover
    • Open link on clicking link icon
    • Mute/umute on clicking mute icon
    • Expand to Full-Screen Mode on clicking the video bubble
  • Full-Screen Mode
    • Play/pause on clicking play/pause buttons
    • Open link on clicking link icon
    • Mute/unmute on clicking mute icon
    • Restart video from the beginning on clicking restart icon
    • Revert back to Standard Bubble Mode on clicking outside the video
  • Additional Aesthetic Features
    • Subtle video bubble bounce animation
    • Subtle glow around video bubble
    • Blur around video screen in Full-Screen Mode
    • Disable scroll in Full-Screen Mode
    • Subtle CSS indicator changes upon hover of icons and video bubbles
